🔒 Browser Security & CORS

Why doesn't my bot work?

Modern browsers block direct API calls to external services like Telegram due to CORS (Cross-Origin Resource Sharing) security policy. This is a browser security feature, not a bug.

Available Solutions:

  1. Browser Extension: Install a CORS-disabling extension like "CORS Unblock"
  2. Local Development: Run the app on localhost (http://localhost)
  3. Different Browser: Try different browsers (some handle CORS differently)
  4. CORS Proxy: Use a CORS proxy service (security risk - use carefully)
  5. Desktop App: Use Telegram Desktop or wrap in Electron
⚠️ Security Note: Disabling CORS reduces browser security. Only do this for trusted sites and development purposes.

Demo Mode: The app falls back to demo mode to showcase features when API calls fail.
