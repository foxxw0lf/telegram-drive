üîß CORS Solutions

Option 1: Browser Extension (Easiest)

  • Install "CORS Unblock" or "Disable CORS" extension
  • Enable the extension for this site
  • Refresh the page and try connecting again

Option 2: Chrome with Disabled Security

  • Close all Chrome windows
  • Run: chrome --disable-web-security --user-data-dir=/tmp/chrome
  • Open this site in the new Chrome window

Option 3: Use Firefox Developer Edition

  • Firefox has more relaxed CORS policies for development
  • Install Firefox Developer Edition
  • Open this site and try connecting

üíª Local Development Setup

Why Localhost Works Better:

Browsers have relaxed CORS policies for localhost, making API calls more likely to succeed.

Setup Instructions:

  1. Download the source code from GitHub
  2. Extract to a local folder
  3. Open terminal/command prompt in that folder
  4. Run a local server:
    • python -m http.server 8000 (Python)
    • npx serve . (Node.js)
    • php -S localhost:8000 (PHP)
  5. Open http://localhost:8000 in your browser

Alternative: Use GitHub Codespaces

  • Fork the repository on GitHub
  • Create a new Codespace
  • Run the app in the cloud development environment
